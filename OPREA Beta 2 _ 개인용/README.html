Otamatone Piano Roll Editor Advanced

Hello, Otamatonists!

OPREA is a Google Spreadsheets–based piano roll editor and web player.

You can easily create, edit otamatone pianoroll scores and share. this score can be customized to your otamatone's interval/range .

No special software, no plugins required.

Once finished, your sheet can be played back instantly through a web-based piano roll player and shared with other users via a simple link.

All you need is a Google account to use Google SpreasSheet.

How To Use

This spreadsheet is an example and cannot be edited directly. To use the editor, you must create your own copy.

1. Make Your Own Copy of the Spreadsheet

· Open the menu: File → Make a copy

· A fully editable version will be created in your Google Drive

2. Make your own setting

· Every Otamatone has different note interval and range.

· Adjust note interval by resizing row heights of ROLL sheet cells.

· Set your otamatone pitch range in the CONFIG sheet.

3. Push the BIG RED Otamatone Button

· It's in PLAYER sheet

· It's the Otamatone Player. You can play & edit the score with it.

· The player request the authority to access this sheet ㅡ for load&write the pianoroll data

· read the secury notice first. And if you think there are no probrems, allow it.

4. I tried hard to make the intuituve UI. So you might not need an explanation

· But if not, you can read whole manual in MANUAL sheet.

사용 방법

 이 스프레드시트는 예시만 보여줄 뿐 수정 권한이 막혀 있습니다. 직접 생성기를 사용하시려면 자신만의 사본을 만드셔야 합니다!

1. 나만의 사본 생성

 · 상단 메뉴 왼쪽 → 파일 클릭 → 사본 만들기

· 위 순서대로 진행하셨다면 자유롭게 수정 가능한 여러분만의 복사본이 개인 구글 드라이브에 생성됩니다.

2. 여러분의 오타마톤에 맞게 악보와 설정을 바꿔주세요

· 오타마톤은 같은 모델이라도 기기마다 음역대와 음 간격이 조금씩 다릅니다

· ROLL시트에서 행간 높이를 조절하여 당신의 오타마톤과 음 간격을 맞추시고

· CONFIG시트에서 음역대와 기타 필요한 설정을 설정하세요.

3. 크고 빨간 오타마톤 버튼을 누르세요

· PLAYER 시트에 있습니다

· 악보 플레이어는 크고 빨간 오타마톤 버튼으로 작동합니다.

· 프로그램이 시트에서 데이터를 가져오고 덮어써야 하기 때문에 접근 권한을 요구합니다.

· 아래 칸의 보안 관련 안내사항을 유심히 읽고, 문제가 없다고 판단되면 실행하세요.

4. 플레이어는 직관적으로 사용하실 수 있게 제작되었습니다.

· 만약 자세한 설명서를 읽고 싶으시다면, MANUAL 시트로 이동해주세요.

Google security warning appeared. Isn't this dangerous?

This warning appears because the app is a personal Apps Script project, not a commercial Google-verified service.

· The pianoroll web player app only accesses your own spreadsheet copy
· No data is sent to external servers
· All source code is fully visible and open for inspection


But you must notice these for security! (Please Read)
 The following points apply not only to this sheet, but to Google Spreadsheets in general.

- Do not store sensitive personal information in the spreadsheet.
 Apps Script runs by reading data from the linked spreadsheet.
Avoid entering private information in any area accessed by the script (e.g. ROLL, CONFIG).

 - Do not create a full copy of spreadsheets made by uncertified third parties.
This original sheet contains only safe and transparent code.
However, spreadsheets modified or redistributed by others may contain malicious Apps Script code.
Always make a copy only from the official sheet.

 - Before use the otamatone player, review the Apps Script code.
If the script contains APIs that access external data or your Google account (such as UrlFetchApp, DriveApp, GmailApp, etc.),
you are strongly advised not to execute it.
If you are unsure, compare it with the original code or ask a trusted developer or AI to review it.

- When sharing your own sheet, keep edit permissions restricted to yourself.
 If others are given edit access, they may freely modify the score, settings, or scripts in your spreadsheet.

- These guidelines are provided to promote safe usage. Any issues resulting from ignoring them are the responsibility of the user’s own judgment.

 How to Share / How to use other's score

1. To share your score
 1) make a copy file of your ROLL sheet. (copy to new spreadsheet)
2) File → Share → Share with others → General Access : anyone with link → Viewer
3) Don't turn the general access permission to the editor.
2. To take other's score :
1) Empty your ROLL sheet score area & song info area.
2) Go to other's ROLL sheet and copy the entire score&song info area. And paste it to your ROLL sheet.
3) If your score area and new score has different note range, use "Otamatone Player" menu → Shift @r tags → Rebuild Cell Event index.
@r means the row number order of notes. Move it as your pitch gap.
악보를 공유 / 사용하는 방법

1. 나의 악보를 공유하는 방법
 1) ROLL 시트를 복사하세요. (다음으로 복사 >> 새 스프레드시트). 이후 시트의 주소를 공유하세요.
2) 파일 → 공유 → 다른 사용자와 공유 → 일반 액세스 : 링크가 있는 모든 사용자 → 뷰어로 설정
3) 편집자 권한을 부여하면 안됩니다. 스프레드시트의 수정 권한은 반드시 자신만 가지고 있어야 합니다.

2. 다른 사람의 악보를 가져오는 방법
1) 내 ROLL 시트의 악보와 곡 정보란을 비웁니다.
2) 배포자의 ROLL 시트에서 악보&곡 정보를 복사합니다.
3) 만일 악보의 음역대가 당신의 악보와 다르다면 우측 위의 "Otamatone Player" 메뉴 → Shift @r tags → Rebuild Cell Event index를 실행하세요.
@r는 노트가 놓이는 행의 번호를 의미합니다. 음높이 차이만큼 숫자를 더하거나 빼서 옮기세요.
