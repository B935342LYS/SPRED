Score Formula

Score per note Track difficulty x pitch accuracy, only when the note is score eligible.
Total score Sum of hit note scores.
Track difficulty Uses custom difficulty from Music Info when set; otherwise defaults to Basic 1.0, Optional 2.0, Extra 3.0.
Combo Easy and Standard: score-eligible Perfect, Ok, or Bad continues the combo. Pro: Bad or Miss resets it.
Easy mode Pitch-only mode. Timing offsets and missing attacks do not block scoring.
No input Silence or unclear pitch does not create a Miss sample and adds no score.
Effect bonus Playing articulations well adds bonus score.
Missing an effect bonus does not create a penalty.
Gliss bonus Track difficulty x 2.5 for each successful 0.25 second gliss interval.
A 1 second gliss can award up to track difficulty x 10.0.
Vib bonus Track difficulty x 5.0 for each successful vibrato segment.
The pitch must stay near the target while showing enough controlled vibration.
Trem bonus Track difficulty x 0.5 for each rewarded tremolo attack.
The first attack starts the streak; bonus begins from the next accepted attack.
Final score Σ(note score) + Σ(Gliss bonus) + Σ(Vib bonus) + Σ(Trem bonus)
