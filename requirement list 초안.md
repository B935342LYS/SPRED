# requirement list (functional)

## 1. 팀 역할 분담


## 2. requirement list

|No.|Requirement|Use Case(s)|
|---|---|---|
|1| 사용자가 ID, 비밀번호, 이름, 전화번호, 이메일 등 필수 정보를 입력할 시 회원 가입할 수 있도록 한다. | 회원 가입 |
|2| 회원이 탈퇴를 선택할 시 해당 회원의 모든 권한과 데이터를 삭제한다. | 회원 탈퇴 |
|3| 관리자 또는 회원이 ID와 비밀번호 입력 시 로그인을 지원한다. | 로그인 |
|4| 회원이 설문 진행 상태(진행 중 / 예정 / 종료) 지정 후 키워드 입력 시 조건에 일치하는 설문 리스트를 출력한다.  | 설문 검색 |
|5| 회원이 특정 설문을 선택 시 새로운 화면을 띄워 상세 정보를 출력한다. 상세 정보는 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각 등으로 구성된다.   회원이 선택한 설문이 "진행 중" 상태일 경우 응답 버튼을 활성화한다. 버튼 클릭 시 응답 화면으로 이동시킨다. | 설문 상세정보 조회 |
|6| 회원이 설문에 직접 응답을 실시할 수 있도록 지원한다. "응답 버튼"을 누를 시 설문을 제출할 수 있다. | 설문 응답 |
|7| 관리자의 설문 정보 등록을 지원한다. 등록시 입력 정보는 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각 등이다. | 설문 등록 |
|8| 관리자가 등록된 모든 설문 리스트 조회할 수 있도록 지원한다. | 설문 리스트 조회 |
|9| 관리자가 모든 설문 리스트 조회 후 특정 항목을 선택하면 팝업 창으로 설문의 상세 내용을 출력한다. | 설문 리스트 조회, 설문 상세 내용 조회 |
|10| 관리자가 모든 설문 리스트 조회 후 특정 항목을 선택하여 삭제를 결정하면 해당 설문 항목을 제거한다. | 설문 리스트 조회, 설문 삭제 |
|11| 관리자가 검색 단위 (1주일 / 1개월 / 1년) 선택 시 해당 기간 범위 이내의 전체 설문 수, 종료된 설문 수, 총 응답 수를 출력한다. | 설문 응답 통계 조회 |
