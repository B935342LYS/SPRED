## 2. Examples Visibility Followup

Examples 목록은 파일 자체 접근 제한이 아니라 목록 노출 제어를 우선한다.

현재 정책:

- `public`
  - access word 없이 표시되는 공개 예제
  - Examples dialog를 열면 즉시 조회된다.
- `extra`
  - access word가 맞을 때 public 목록과 함께 추가 표시되는 테스트 / 추가 예제
- `hidden`
  - DB에는 남기지만 앱 목록에는 표시하지 않는 예제

프론트엔드 동작:

- File > Examples를 누르면 public list가 자동 조회된다.
- access word 없이 별도 목록 요청을 해도 invalid error를 띄우지 않는다.
- access word 입력 후 누르는 버튼 문구는 `Load Extra List`로 유지한다.
- 잘못된 non-empty access word는 기존 score와 기존 list를 유지하면서 오류를 표시한다.

관련 구현:

```text
regression-code/src/app/examples/
regression-code/dev/supabase_get_example_manifest_edge_function.ts.txt
regression-code/dev/example_catalog_generator.html
```

`example_catalog_generator.html`은 DB row 생성 시 list visibility를 선택하고, 현재 구조에 맞춰 `access_group`과 `is_published` 값을 생성한다.

## 3. Score Comment Limit

Score comment 길이 제한은 기존 100자에서 500자로 확장했다.

변경된 기준:

- `musicData.comment`는 문자열이어야 한다.
- 최대 길이는 500자이다.
- 저작권 프리 음악 출처 표기처럼 여러 줄 credit이 필요한 경우를 수용한다.

관련 구현:

```text
regression-code/src/core/score/score_limits.ts
regression-code/src/core/score/score_validate.ts
regression-code/src/core/score/types.ts
regression-code/src/app/app_view_actions.ts
regression-code/dev/test_score.ts
```

`index.html`의 Details comment textarea도 `maxlength="500"`으로 맞췄다.

## 4. Details Comment Height

Details 내부 Comment 입력창은 사용자가 드래그로 크기를 조절하기 전에도 일반적인 500자 credit 문구를 볼 수 있도록 기본 높이를 키웠다.

현재 HTML:

```html
<textarea class="details-comment-input" maxlength="500" rows="10"></textarea>
```

이 변경은 active implementation과 GitHub Pages test deployment copy 모두에 반영했다.

## 5. Bundled Normal Score Layout

기본 제공 layout preset을 추가했다.

파일:

```text
regression-code/src/assets/templates/normal-score-layout-preset.ts
regression-code/src/assets/templates/normal-score-layout-preset.json
```

프리셋:

- slot: `1`
- display name: `Normal Score`
- instrument preset: `otamatone-basic`
- range: `C3`부터 `C6`
- row 구성: note row만 포함
- row height: 모든 note row `21px`
- gap row 없음

`loadLayoutPresetSlotFromLocalStorage()`는 해당 악기 프리셋의 slot 1에 사용자 저장값이 없을 때 bundled Normal Score preset을 반환한다.
사용자가 slot 1에 직접 저장한 값이 있으면 localStorage 값이 우선한다.
