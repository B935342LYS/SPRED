SPRED Manual

SPRED는 악보 JSON을 만들고, 불러오고, 재생하며, practice mode로 연습하기 위한 웹 앱입니다.

Files

  • JSON Load: 로컬 Score JSON 파일을 불러옵니다.
  • JSON Download: 현재 악보를 JSON 파일로 저장합니다.
  • Local Save / Local Load: 현재 브라우저의 local storage에 악보를 저장하거나 다시 불러옵니다.
  • Examples: 임시 access word를 입력해 원격 예제 악보 목록을 불러옵니다.

Playback

Play, Stop, seek bar로 재생 위치를 제어합니다. Speed는 화면 열 너비만 바꾸며 실제 재생 시간은 바꾸지 않습니다.

Edit Mode

Edit Mode에서는 활성 track과 선택한 입력 도구에 따라 score cell rawText를 수정합니다. Practice mode 중에는 편집과 파일 교체가 잠깁니다.

YouTube Sync

YouTube mode는 공식 embedded player를 follower로 사용합니다. 앱은 영상이나 음원을 저장, 캐시, 추출, 재배포하지 않습니다.

Practice Mode Beta

마이크 입력을 사용해 pitch와 timing을 판정하는 연습 보조 기능입니다. 조용한 장소에서 마이크를 사용하세요.
