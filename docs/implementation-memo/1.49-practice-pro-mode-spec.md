# 1.49 Practice Pro Mode Spec

## 1. Scope

이번 메모는 practice mode의 판정 강화 옵션인 Pro Mode 명세 결정을 기록한다.

Pro Mode는 아직 구현하지 않았고, `docs/3.2-karaoke-game-mode-spec.md`에 후속 구현 기준을 추가한 상태이다.

## 2. Motivation

현재 Standard practice 판정은 프렛리스 악기와 마이크 입력 환경을 고려해 비교적 관대하다.

이 정책은 초보자와 일반 연습에는 적합하지만, 더 엄밀한 연주를 원하는 사용자에게는 목표가 약할 수 있다.

따라서 기본 판정을 유지하면서, 사용자가 직접 선택할 수 있는 세션 단위 강화 모드를 추가한다.

## 3. Decisions

- 이름은 `Pro Mode`로 둔다.
- 내부 상태 후보는 `PracticeJudgeMode = "standard" | "pro"`이다.
- 기본값은 `standard`이다.
- Pro Mode 값은 score JSON에 저장하지 않는다.
- Pro Mode 값은 localStorage에 저장하지 않는다.
- Pro Mode 값은 브라우저 페이지 세션 동안만 유지한다.
- `Sync` 오른쪽에 `Pro` toggle button을 둔다.
- practice off / ready 상태에서만 toggle 가능하다.
- preparing / countdown / playing / paused / finished 상태에서는 toggle을 잠근다.
- Pro Mode UI는 red accent를 사용하되, error 상태와 혼동되지 않도록 강한 오류 색 전체 배경은 피한다.
- result dialog에는 가능하면 `Mode: Standard` 또는 `Mode: Pro`를 표시한다.

## 4. Judge Thresholds

Standard pitch threshold는 기존 구현을 유지한다.

| Mode | Perfect | Ok | Bad | Miss |
| --- | ---: | ---: | ---: | ---: |
| Standard | `< 50 cent` | `< 100 cent` | `< 200 cent` | `>= 200 cent` |
| Pro | `< 30 cent` | `< 60 cent` | `< 100 cent` | `>= 100 cent` |

Standard timing threshold는 기존 구현을 유지한다.

| Mode | No timing text | Early/Late | Bad downgrade | Miss downgrade |
| --- | ---: | ---: | ---: | ---: |
| Standard | `< 80ms` | `80-149ms` | `150-249ms` | `>= 250ms` |
| Pro | `< 50ms` | `50-99ms` | `100-149ms` | `>= 150ms` |

## 5. Combo Policy

Standard는 현재 연습 친화 정책을 유지한다.

```text
scoreEligible Perfect / Ok / Bad -> combo +1
Miss -> combo reset
scoreEligible false / no input -> combo unchanged
```

Pro Mode는 더 엄격한 성취 목표를 제공한다.

```text
scoreEligible Perfect / Ok -> combo +1
Bad / Miss -> combo reset
scoreEligible false / no input -> combo unchanged
```

Combo는 여전히 점수 배율에 영향을 주지 않는다.

## 6. Unchanged Policies

Pro Mode에서도 다음 정책은 유지한다.

- pitch class 허용
- target 기반 octave correction 제거 상태
- harmonic correction 제거 상태
- hysteresis lock 제거 상태
- no-onset 직접 Bad/Miss 강등 없음
- score formula 동일
- effect bonus 정책 동일
- gliss grace 동일
- vib 판정 동일
- scoreEligible false sample은 overlay와 score summary에 반영하지 않음
- no input은 Miss를 만들지 않음

## 8. Verification Plan

필수 단위 검증:

- Standard pitch threshold는 기존 결과를 유지한다.
- Pro pitch threshold는 `30/60/100 cent` 기준으로 label을 만든다.
- Standard timing threshold는 기존 결과를 유지한다.
- Pro timing threshold는 `50/100/150ms` 기준으로 timing result와 label downgrade를 만든다.
- Standard에서 Bad는 combo를 이어간다.
- Pro에서 Bad는 combo를 초기화한다.
- Pro Mode에서도 Miss는 scoreContribution `0`이다.
- Pro Mode에서도 scoreEligible false sample은 overlay/summary 대상이 아니다.

브라우저 수동 검증:

- Pro toggle이 `Sync` 오른쪽에 보인다.
- Pro toggle은 off / ready 상태에서만 변경된다.
- playback 중 Pro toggle은 disabled이다.
- Pro Mode가 켜져 있으면 practice panel red accent가 보인다.
- 새로고침하면 Pro Mode가 standard로 돌아간다.
- result dialog에 mode 표시를 추가한 경우 실제 mode와 일치한다.
